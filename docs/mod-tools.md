# Moderator Tools – ZippyHunt

This document outlines the tools and commands available to moderators for reviewing, editing, and deploying quests or events.

---

## 🧰 Moderator Actions

| Action                  | Description                                    |
|-------------------------|------------------------------------------------|
| Approve Quest           | Publishes quest to ZippyQuest                  |
| Reject Quest            | Marks submission as declined                   |
| Flag for Review         | Sends to ZQ Codex for further analysis         |
| Merge with Existing     | Combines duplicate community quests            |
| Edit Location           | Adjusts map pin location                       |
| Add Sponsor             | Tags quest with business partner info          |
| Assign Event            | Queues quest for use in an upcoming event      |

---

## 🧪 Draft Tools

| Tool           | Use Case                                  |
|----------------|-------------------------------------------|
| “Quick Build”  | Auto-build simple quest from checklist     |
| “Lore Linker”  | Tie new quests into a narrative arc        |
| “Badge Picker” | Suggests badge awards based on type/class |

---

## 🔐 Permissions

| Role        | Access                       |
|-------------|------------------------------|
| Admin       | Full backend + publishing     |
| Regional    | Area-specific quests/events   |
| Volunteer   | Read-only review, comment only|
